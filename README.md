Requisitos do Sistema
• O aplicativo deve possuir três níveis de dificuldade: Fácil, Médio e Difícil. O usuário deve escolher um
nível antes de iniciar o quiz.
• Após escolher o nível, o aplicativo deve mostrar apenas perguntas daquele nível.
• Criar uma classe ou objeto chamado Pergunta contendo pelo menos: texto da pergunta, resposta
correta e nível da pergunta.
• O grupo pode opcionalmente adicionar uma lista de opções caso queira implementar múltipla escolha.
• O usuário deve poder responder digitando a resposta em um TextField ou selecionando uma opção.
• Ao responder, o aplicativo deve validar a resposta e mostrar se está correta ou incorreta.
• O aplicativo deve manter e exibir a pontuação do usuário.
• Depois de responder, o usuário deve poder ir para a próxima pergunta.
Desafios Extras (Opcional)
• Embaralhar a ordem das perguntas.
• Embaralhar as opções de resposta.
• Adicionar cores para indicar acerto ou erro.
• Impedir avançar sem responder.
• Permitir reiniciar o quiz.
• Mostrar a porcentagem de acertos.
